# Opinion-Dynamics / Sociophysics Models with Bistability

## Executive summary

Bistability in opinion dynamics is almost never produced by purely linear averaging on a row-stochastic matrix (which is what our current substrate is): such systems are contractions and admit a unique fixed point, exactly as we proved. Across the sociophysics literature, bistability and hysteresis are generated by one of three sharply identifiable nonlinear mechanisms — (i) a saturating/sigmoidal coupling (`tanh`) gated by an attention/sensitivity parameter that creates a pitchfork or saddle-node in the symmetric state (Bizyaeva–Franci–Leonard family, Wang–Rong–Cheng), (ii) a discontinuous/threshold interaction rule such as bounded confidence (Hegselmann–Krause, Deffuant–Weisbuch) or commit-then-flip (q-voter, Sznajd, naming game with committed minorities), or (iii) zealots/stubborn agents providing a state-dependent external field on top of an otherwise linear rule. For our codebase, mechanism (i) is the cheapest graft — it requires only replacing the linear Γ-weighted pool in `population.py` with a sensitivity-gated saturating update, leaves the Bayesian `inference.py` untouched, and immediately yields a pitchfork at a tunable threshold.

### Bizyaeva, Franci & Leonard 2023 — Nonlinear Opinion Dynamics With Tunable Sensitivity

**BibTeX**
```bibtex
@article{bizyaeva2023nonlinear,
  title   = {Nonlinear Opinion Dynamics With Tunable Sensitivity},
  author  = {Bizyaeva, Anastasia and Franci, Alessio and Leonard, Naomi Ehrich},
  journal = {IEEE Transactions on Automatic Control},
  volume  = {68},
  number  = {3},
  pages   = {1415--1430},
  year    = {2023},
  doi     = {10.1109/TAC.2022.3159527}
}
```

**Code.** No official repository found on the authors' Princeton NAOMI lab page or Bizyaeva's homepage. Several third-party reimplementations exist on GitHub but I did not verify a canonical one. Treat as "no working code we can clone today."

**Nonlinearity giving bistability.** `dx_i/dt = -d x_i + u * S(Σ_j A_ij x_j + b_i)` where `S` is an odd saturating function (`tanh`); the attention/sensitivity gain `u` passes through a pitchfork bifurcation of the neutral fixed point `x=0` at a critical `u*` set by the leading eigenvalue of the signed network — above `u*` the symmetric state is destabilized and two opposite-sign opinion equilibria appear.

**Graft.** This is the closest fit to our substrate. In `population.py`, replace the current linear pooled mean `μ_i ← Σ_j Γ_ij μ_j` with `μ_i ← μ_i + dt · (-d μ_i + u · tanh(Σ_j Γ_ij (μ_j − μ_i)))`, exposing `u` as a config knob. `inference.py` stays as-is (likelihood update on observations remains Gaussian-conjugate); `trust.py` continues to feed `Γ`; `resource.py` and `policy.py` are untouched. Bistability appears as soon as `u` crosses the spectral threshold of `Γ`. This is the minimum-disruption path to a real pitchfork in the model.

### Wang, Rong & Cheng 2016 — Bistability and multistability in opinion dynamics models

**BibTeX**
```bibtex
@article{wang2016bistability,
  title   = {Bistability and multistability in opinion dynamics models},
  author  = {Wang, Shaoli and Rong, Libin and Wu, Jianhong},
  journal = {Applied Mathematics and Computation},
  volume  = {289},
  pages   = {388--395},
  year    = {2016},
  doi     = {10.1016/j.amc.2016.05.030}
}
```

**Code.** None found. Paper is purely analytical (ODE bifurcation analysis).

**Nonlinearity giving bistability.** Zealot/committed-believer term added to a two-opinion mean-field ODE: `dx/dt = -x + (1-x-c) * f(x) + c`, where `c` is the committed fraction and `f` is a quadratic/cubic recruitment rate. Saddle-node bifurcation appears as `c` is varied; for an interval of `c` two stable opinion equilibria coexist.

**Graft.** Requires (a) introducing a "committed" agent class — either an absorbing flag in `population.py` or a small set of agents whose `inference.py` posterior update is replaced by a clamp on `μ_i`, and (b) ensuring the pooling/coupling has a recruitment-style nonlinearity (otherwise zealots only shift the unique fixed point, they don't create a second one — this is a known result for our linear pool). Minimal-edit path: `population.py` (clamp subset) + replace the linear pool with the saturating one from Bizyaeva above. By itself, adding zealots to our current linear substrate is **insufficient** for bistability.

### Deffuant, Neau, Amblard & Weisbuch 2000 — Mixing beliefs among interacting agents (bounded confidence)

**BibTeX**
```bibtex
@article{deffuant2000mixing,
  title   = {Mixing beliefs among interacting agents},
  author  = {Deffuant, Guillaume and Neau, David and Amblard, Fr{\'e}d{\'e}ric and Weisbuch, G{\'e}rard},
  journal = {Advances in Complex Systems},
  volume  = {3},
  number  = {01n04},
  pages   = {87--98},
  year    = {2000},
  doi     = {10.1142/S0219525900000078}
}
```

**Code.** Implemented in NDlib (`GiulioRossetti/ndlib`, Python; last release Sept 2021 — usable but quiet). Many third-party Python reimplementations on GitHub (`dmpalyvos/opinions-research`, etc.).

**Nonlinearity giving bistability.** Hard threshold (indicator function) on opinion distance: agents `i,j` average their opinions only if `|x_i − x_j| < ε`; otherwise zero interaction. Sub-critical `ε` produces fragmentation into ≥2 persistent clusters; supra-critical `ε` produces consensus. Hysteresis is **structural** (in cluster count) not in a single order parameter, but the multi-cluster regime is genuinely multistable in initial conditions.

**Graft.** Replace the row-stochastic `Γ_ij` in `population.py` with a confidence-gated `Γ_ij · 𝟙[|μ_i − μ_j| < ε]`, renormalised per row. `trust.py` then learns precisions only across agents currently within confidence — this matches the "attentional siloing" we already saw empirically in notebook 08. This is the single change with the strongest theoretical pedigree, but it produces *fragmentation-multistability*, not the smooth pitchfork we'd ideally want for a paradigm-transition narrative.

### Castellano, Muñoz & Pastor-Satorras 2009 — Nonlinear q-voter model

**BibTeX**
```bibtex
@article{castellano2009nonlinear,
  title   = {Nonlinear $q$-voter model},
  author  = {Castellano, Claudio and Mu{\~n}oz, Miguel A. and Pastor-Satorras, Romualdo},
  journal = {Physical Review E},
  volume  = {80},
  number  = {4},
  pages   = {041129},
  year    = {2009},
  doi     = {10.1103/PhysRevE.80.041129}
}
```

**Code.** `robertjankowski/real-q-voter` (Python, mature, 36 commits — usable). Also implemented in `GiulioRossetti/ndlib`. Both clone-and-run today.

**Nonlinearity giving bistability.** An agent flips its ±1 opinion only if **all `q` randomly chosen neighbours** agree; otherwise it flips with small probability `ε` (noise). The "unanimity of `q`" rule is the nonlinearity: for `q ≥ 4` the model exhibits a discontinuous (first-order) phase transition and hysteresis loops between an ordered and a disordered phase. Doniec, Sznajd-Weron & Vazquez (arXiv:2511.17837, 2025) recently showed **double hysteresis** under asymmetric `q₊`, `q₋`.

**Graft.** Hardest fit. The mechanism is genuinely discrete (binary spin, unanimity) and does not map cleanly onto continuous `μ`. To port: discretize `μ_i` into a sign or use a multi-state quantization in `policy.py`, and replace the social term in `population.py` with a q-unanimity check on the sampled q-neighbour subset (weighted by `Γ`). This is a substantial rewrite and abandons our Gaussian-conjugate substrate.

### Xie, Sreenivasan, Korniss, Zhang, Lim & Szymanski 2011 — Naming game with committed minorities / tipping points

**BibTeX**
```bibtex
@article{xie2011social,
  title   = {Social consensus through the influence of committed minorities},
  author  = {Xie, J. and Sreenivasan, S. and Korniss, G. and Zhang, W. and Lim, C. and Szymanski, B. K.},
  journal = {Physical Review E},
  volume  = {84},
  number  = {1},
  pages   = {011130},
  year    = {2011},
  doi     = {10.1103/PhysRevE.84.011130}
}
```

**Code.** No canonical repo. Reimplementations exist (Baronchelli's group has standalone naming-game code in C/Python scattered across labs); not a clean clone target.

**Nonlinearity giving bistability.** Listener–speaker word-exchange dynamics with a fraction `p` of committed agents who never update. Below `p_c ≈ 0.10`, consensus on the minority view takes time exponential in `N` (effective bistability); above `p_c`, consensus is reached in logarithmic time — saddle-node-like bifurcation in the mean-field reduction over (single-word, mixed-word) population fractions.

**Graft.** Same as Wang-Rong above: needs a committed-agent flag in `population.py`. But naming-game state space is *vocabulary* (sets of labels), which doesn't map onto our Gaussian `μ`. Could be ported by treating "committed" as a clamp on a discrete paradigm-label drawn from the sign of `μ`, but this changes our state representation, so it's a heavier graft than Bizyaeva.

### Galam 2008 — Sociophysics review (majority-rule with contrarians / inflexibles)

**BibTeX**
```bibtex
@article{galam2008sociophysics,
  title   = {Sociophysics: A review of Galam models},
  author  = {Galam, Serge},
  journal = {International Journal of Modern Physics C},
  volume  = {19},
  number  = {03},
  pages   = {409--440},
  year    = {2008},
  doi     = {10.1142/S0129183108012297},
  eprint  = {0803.1800},
  archivePrefix = {arXiv}
}
```

**Code.** None official. Many ad-hoc Python implementations exist; none authoritative.

**Nonlinearity giving bistability.** Local majority rule applied iteratively over random subgroups of size `r` (typically 3), with a fraction `a` of contrarians (always vote against local majority) or `α` of inflexibles (never update). The discrete majority operator is the nonlinearity — its mean-field map `f(x)` is a cubic-like polynomial with two stable fixed points separated by an unstable tipping point. Inflexibles produce hysteresis as `α` is ramped.

**Graft.** Same family as q-voter — requires discretization of `μ` into a vote and a majority operator in `population.py`. Not natural for our continuous-θ substrate. Useful primarily as a reference for "inflexibles ≈ zealots ⇒ shifted but unique fixed point unless you also have a nonlinear social rule."

### Nowak, Szamrej & Latané 1990 — Social impact theory (cellular automaton)

**BibTeX**
```bibtex
@article{nowak1990private,
  title   = {From private attitude to public opinion: A dynamic theory of social impact},
  author  = {Nowak, Andrzej and Szamrej, Jacek and Latan{\'e}, Bibb},
  journal = {Psychological Review},
  volume  = {97},
  number  = {3},
  pages   = {362--376},
  year    = {1990},
  doi     = {10.1037/0033-295X.97.3.362}
}
```

**Code.** None. Recent reanalyses (arXiv:2503.16721, 2025) exist in Fortran/Python but no canonical release.

**Nonlinearity giving bistability.** Each agent's impact on a target scales as `Σ_j s_j / d_ij^α` (distance-decayed sum) and the target flips sign only if the impact exceeds its own "strength"-weighted self-support — i.e. a sign function on a weighted sum, which is the hard threshold giving phase transitions. Produces metastable minority clusters that persist indefinitely (effective bistability) on 2D lattices.

**Graft.** Closest to a "field-theoretic" version of our `trust.py × population.py` coupling. Could be implemented by replacing the row-stochastic `Γ` with an un-normalised impact kernel and adding a sign/threshold flip rule in `policy.py`. Less directly relevant than Bizyaeva because the impact function is lattice/distance-specific.

## Cross-cutting note on our substrate

Three of the seven families above (Wang–Rong, naming-game, Galam-inflexible) rely on **committed/zealot agents on top of an already-nonlinear social rule**. Adding zealots to our current linear pool produces a shifted but still unique fixed point — this is exactly the failure mode we hit in notebook 13. The Bizyaeva–Franci–Leonard saturating coupling is the only graft that produces bistability with a single, well-localised edit in `population.py` while preserving the rest of our Bayesian machinery.
